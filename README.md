Aim:
To study and implement list operations in Python using built-in functions and methods such as max() and append().

Theory:
A list in Python is a mutable data structure that allows storing multiple elements in a single variable. Lists can contain different types of data and can be modified after creation. Python provides several built-in functions and methods that make list operations easier and more efficient. The max() function is used to identify the largest element in a list, while the append() method allows adding a new element to the end of an existing list. These features make lists useful for organizing, managing, and manipulating data in Python programs.
